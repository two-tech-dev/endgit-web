# Security Policy

## Supported Versions

Currently, only the `main` branch is actively supported with security updates. 

## Reporting a Vulnerability

If you discover a security vulnerability within EndGit Web or any EndGit components, please DO NOT open a public issue.

Instead, please send an email to the repository maintainers or use GitHub's "Report a vulnerability" feature under the Security tab of the repository.

All security vulnerabilities will be promptly addressed, and we will work with you to test and release the patch.
